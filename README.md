# Frontend Demo

This repository contains a minimal frontend-only React project scaffolded with Vite and TypeScript, plus unit tests using Vitest and Testing Library.

Getting started

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Run tests

```bash
npm test
```

Project layout (important parts)

- `src/components` — shared UI components
- `src/pages` — page-level components
- `src/hooks` — reusable hooks
- `src/services` — API or network logic (mocked here)
- `src/utils` — small pure helpers (unit tested)
